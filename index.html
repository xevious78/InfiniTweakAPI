    Introduction

    Welcome to the InfiniTweak API !

    API_VERSION = 0.1

    Response Structure

    Successful query without result 

    {
    "API" : <STR:API_VERSION>,
    "SUCCESSFUL" : true
}

    Successful query with result 

    {
    "API" : <STR:API_VERSION>,
    "SUCCESSFUL" : true,
    "RESULT" : <OBJ:RESULT>
}

    Unsuccessful action

    {
    "API" : <STR:API_VERSION>,
    "RESULT" : false,
    "REASON" : <STR:ERROR>
}

    Status Codes

    Code Meaning Reason
    200 Success

    Callback Structure

    Callback Structure without details

    {
    "API" : <STR:API_VERSION>,
    "CALLBACK" : <STR:CALLBACK_MESSAGE>
}

    Callback Structure with details

    {
    "API" : <STR:API_VERSION>,
    "CALLBACK" : <STR:CALLBACK_MESSAGE>,
    "DETAILS" : <OBJ:DETAILS>
}

    REST API Requests

    Engine : Init

    Create the I/O devices.

    HTTP Request

    GET http://InfiniTweak/api/engine/init

    Result

    None

    Error

    Engine : Start

    Start the audio rendering process

    HTTP Request

    GET http://InfiniTweak/api/engine/start

    Result

    None

    Error

    Error Reason
    ENGINE_CLOSED The engine is closed

    Engine : Pause

    Pause the audio rendering process

    HTTP Request

    GET http://InfiniTweak/api/engine/pause

    Result

    None

    Error

    Error Reason
    ENGINE_CLOSED The engine is closed

    Engine : Close

    Doesn’t work

    Stop the engine and remove all the I/O devices.

    HTTP Request

    GET http://InfiniTweak/api/engine/close

    Result

    None

    Error

    Error Reason
    UNKNOWN
